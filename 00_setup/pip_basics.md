## What is pip?
`pip` is Pythonâ€™s tool to **install, update, and remove packages**.

---

## Installing Packages
To install a package:

```bash
pip install package_name
```
Example
```bash
pip install flask
```
## Upgrading Packages
To upgrade an existing package:

```bash
pip install --upgrade package_name
```

## Uninstalling Packages
To remove a package:

```bash
pip uninstall package_name
```

## Listing Installed Packages
To see all packages installed in your environment:

```bash
pip list
```
